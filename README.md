# CELPIP Practice Website

A personal CELPIP-General practice site built by Akash for Listening, Reading, Writing, and Speaking practice.

- Reading: multi-part passages with exam-style layout and timers.
- Listening: one-play audio with locked transcript until after submission.
- Writing: task 1 email with 27-minute timer and live word count.
- Speaking: 8 task types with preparation and response timers.

Live site: https://starlit-travesseiro-cd931f.netlify.app/
